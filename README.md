## 🏛️ Project Architecture
The project follows a "Cipher Mode" design language: dark, high-contrast, and focused on authority.

- **`index.html`**: The primary gateway featuring the Hero Fold, Engine details, and the War Room dashboard.
- **`style.css`**: The design system. Includes glassmorphism, scanlines, noise overlays, and vibrant cyan/green accents.
- **`main.js`**: Orchestrates GSAP animations (ScrollTrigger, staggered entrances) and form handling logic.
- **`thank-you.html`**: The destination after the "Hero Protocol" is initiated, featuring a simulated voice confirmation from JAX.
- **`assets/images/`**: High-fidelity AI-generated assets for JAX's portrait and system screenshots.
